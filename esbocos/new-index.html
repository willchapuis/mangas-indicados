Bem-vindo ao seu Registro de Obras

Acompanhe, registre e organize suas leituras

Este projeto pessoal permite que você mantenha um controle completo sobre suas leituras, indicações e atualizações de obras orientais como mangás, manhwas e manhuas. Use os links acima para navegar entre as páginas.

Como funciona?

Você pode visualizar todas as indicações recebidas, cadastrar novas, acompanhar obras em andamento e registrar detalhes sobre cada leitura.

Foco em autonomia

O site funciona localmente, com estrutura pensada para expansão futura via React ou Django. Os dados podem ser integrados via arquivos ou backend leve.
